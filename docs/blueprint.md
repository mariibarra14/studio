# **App Name**: TicketVerse

## Core Features:

- User Registration: Allow new users to register with personal details, including profile photo upload. Clear validation is enforced.
- User Login: Enable users to log in securely using email and password.
- Profile Management: Users can view and edit their profile details, including password changes.
- Payment Methods: Display and manage stored payment methods (card details).
- Activity History: Show a chronological log of user activities.

## Style Guidelines:

- Primary color: Deep purple (#6246EA) to convey a sense of sophistication and exclusivity, aligning with high-value tickets.
- Background color: Very light gray (#F5F3FF), a desaturated shade of purple providing a clean, modern backdrop.
- Accent color: Lavender (#B8B8FF) for highlights and interactive elements, complementing the primary purple.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable style.
- Use minimalist icons from a set like Phosphor Icons, maintaining a consistent stroke weight for a clean, modern aesthetic.
- Employ a responsive grid layout with Shadcn/UI components to ensure optimal viewing on all devices. Utilize spacing and padding to create visual balance.
- Incorporate subtle transitions and animations for button hovers and page transitions to enhance user engagement without being distracting.
